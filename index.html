Criticism of Visualization

Case Study 1: Visualizing the History of Pandemics

This visualization tells tahe historical series of pandemics that have occurred in the world, along with information about the name of the pandemic, the pandemic period, and the death toll number. This visualization was created by Johns Hopkins University in 2021.

Appreciation

Visualization Is Easy to Understand

The purpose of this visualization is to convey the pandemics that have occurred, narrated by a specific timeline, and this story is easily understood and clearly displayed in this visualization. In other words, this visualization successfully conveys the purpose of the existing data.

Criticism and Suggestions

3D Display is Less Effective and Causes Confusion

The virus icon attempts to show that the larger the size of the bacteria, the higher the death toll in that pandemic. However, if we want to quickly see the data to find which pandemic has the highest death toll, we might choose the Spanish Flu as the pandemic with the highest death toll, when in fact the highest is the Black Death.

The Use of Attributes Can Still Be Improved

More than just using colors for each virus to show the differences between each pandemic, maybe I think it would be more informative if the virus icon is not represented with a different color but uses a virus icon that is close to the actual virus.




Case Study 2: LGBTQ+ Representation on TV

This visualization shows the representation of LGBTQ+ in media, and, when viewed, this data is represented in the form of a multilayer pie chart. This data shows LGBTQ+ representation based on several layers.

  • First Layer is the Network
  • Second Layer is Genre
  • Third Layer is Type of Representation
  • Fourth Layer LGBTQ+ Involvement as an actor
  • Five Layer is Number of Season
  • Six Layer is Running Years

Criticism and Suggestions


Varied But Not Informative Visualization

This visualization is difficult to deduce and can hardly be concluded accurately. This data does not show percentage data in the pie chart section, so the only conclusions that can be drawn are the largest and smallest data percentages based on the category, e.g., LGBTQ+ representation on TV by the network.

The Colors Used Are Ineffective

This visualization uses sequential colors as if the data scale is ordinal. Still, the data shown actually uses a nominal scale, e.g., drama, comedy, fantasy, etc., which is a film genre.

Inconsistent Use of Color

The first circle layer is a uniformly colored network, but then the other layers are divided into different colors.

Suggestion

This data may be more effective if it is broken down into categories, or it may be overlaid between categories. Interactive visualizations can make these visualizations more informative.
